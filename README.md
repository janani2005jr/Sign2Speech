# 🤟 Sign2Speech

A real-time Sign Language Recognition system that detects hand gestures using **MediaPipe** and **OpenCV** and converts them into speech through a **Flask** web application.

---

## 📌 Features

- 🎥 Real-time webcam detection
- 🤚 Hand gesture recognition
- 🔊 Speech output
- 🌐 Flask-based web interface
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- Python
- Flask
- OpenCV
- MediaPipe
- HTML
- CSS
- JavaScript

---

## 📂 Project Structure

```
Sign2Speech/

│── speech.py
│── requirements.txt
│── templates/

```

---

## 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/Sign2Speech.git
cd Sign2Speech
pip install -r requirements.txt
python app.py
```
